# Iris_Dataset_classification-Model_Comparison
This script performs classification on the Iris dataset using multiple algorithms and compares their performance using 10-fold cross-validation.

# Iris Dataset Classification

A machine learning project comparing multiple classification algorithms on the Iris dataset.

## 📊 Algorithms Compared
- Logistic Regression (LR)
- Linear Discriminant Analysis (LDA)
- K-Nearest Neighbors (KNN)
- Decision Tree (CART)
- Gaussian Naive Bayes (NB)
- Support Vector Machine (SVM)

## 🚀 Usage
```bash
python iris_classification.py
